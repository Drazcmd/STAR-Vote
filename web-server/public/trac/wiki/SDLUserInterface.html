SDL User Interface

VoteBox? can be launched with a minimalist UI which makes use of the Simple DirectMedia? Layer. By using this view, VoteBox? can be run without depending on a window manager (such as X.org). The library required for this is SDL_wrap, a custom wrapper of SDL; which makes use of SWIG for marshalling between Java and C. SDL_wrap is only ment to be used on Linux systems running without a window manager, although a .dll is included (and will build) known issues exist on Windows.

To use the SDL UI, the VIEW_IMPLEMENTATION must be set to SDL in the appropriate .conf file: 

VIEW_IMPLEMENTATION
SDL

Additionally, libSDL_wrap.so must be in your Java library path, or passed as such on the command line: 

java -Djava.library.path=my/path/to/lib/SDL_wrap/folder -jar ...
